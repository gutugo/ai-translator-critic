## Юнит-тесты (Vitest) — 30 тестов

1. Откройте терминал в папке проекта  
2. Выполните команду:
   ```bash
   npm run test
3. Если команда не найдена — один раз выполните:
   npm install

## UI-тесты (Cypress) — 30 тестов

1. Откройте терминал в папке проекта
2. Запустите приложение (оно откроется на localhost):
   npm run dev:client
   Дождитесь строки Local: http://localhost:5000/ или 5001/
3. Откройте второй терминал (не закрывая первый!)
4. Выполните:
   npx cypress run --config baseUrl=http://127.0.0.1:5001
   (если приложение запустилось на порту 5000 — пишите 5000 вместо 5001)
